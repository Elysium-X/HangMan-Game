## Tech Stack

- **Framework**: React (function components with hooks)
- **Language**: TypeScript with TSX (`.tsx` React components)
- **Build Tool / Dev Server**: Vite
- **Styling**: CSS & CSS Modules
- **Tooling**: ESLint + TypeScript-ESLint, TypeScript compiler

## Learning Goal

The main goal of this project is to **practice TypeScript (TSX) with React** by building a small Hangman game.  
It focuses on:

- Managing component state with React hooks
- Typing props and component state with TypeScript
- Working with TSX-based UI components and basic styling
